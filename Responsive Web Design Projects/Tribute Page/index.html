History of Cascading Style Sheets (CSS)

Timeline of CSS

  • December 1996
    W3C published the first standard for CSS
  • June 1999
    CSS3 drafts for color profiles, multi-column layout, and paged media
  • May 2003
    Dave Shea launches CSS Zen Garden
  • October 2006
    Qt Style Sheets, CSS-like styles for GUIs
  • May 2008
    Microsoft and Monotype open for EOT format
  • August 2009
    LESS CSS pre-processor
  • June 2011
    MapCSS, CSS-like styles for maps
  • November 2016
    CSS Intro course, a MOOC on W3Cx
  • December 2016
    20th anniversary of CSS level 1
  • January 2017
    IDPF and W3C join to develop e-book standards
